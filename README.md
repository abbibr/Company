<h1>Laravel Company Project</h1>

Admin esa barcha nomerlarni va qaysi kliyent tomonida kirtilganini koradi. Kliyentlar hech qanday malumotni ochira olmaydi, admin esa barcha malumotlarni ochirishi mumkin.
